O que você irá estudar?

Você vai estudar como criar web sites em HTML, adequados às normas passadas pela W3C, também conhecidas como padrões web(webstandards).

Por que estudar isso?

Para desenvolver sites capazes de serem lidos da mesma forma em qualquer dispositivo, conseguir trabalhar em uma equipe de desenvolvimento,, sendo capaz de criar aplicações web.

Para quem serve este conhecimento?

Essa pergunta é importante. Existem dois grupos de profissionais que trabalhan com HTML: aqules que usam de forma direta e os que usam de forma indireta. Aqules que usam de forma direta, são aqueles cuja a profissão depende de saber usar HTML, como os desenvolvedores web, web designers, designers que trabalham com UX e etc. Já aqueles que usam de forma indireta, são aqueles que não dependemdo conhecimento de HTML, mas que podem precisar saber algumas coisas para facilitar seu trabalho.

Onde usar este conhecimento?

Voce pode fazer uso do conhecimento de HTML em diversas situações,por exemplo: • Criação de sites institucionais ou blogs; • Desenvolvimento de sistemas web; • Estruturação de e-mails marketing; • Produção de conteúdo para plataformas digitais; • Otimização de interfaces para diferentes dispositivos.
